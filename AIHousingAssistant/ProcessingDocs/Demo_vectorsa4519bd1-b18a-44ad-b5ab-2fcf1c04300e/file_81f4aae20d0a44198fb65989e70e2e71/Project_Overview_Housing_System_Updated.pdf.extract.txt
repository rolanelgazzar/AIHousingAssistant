PROJECT NAME
Housing Management System
PROBLEM PROJECT WILL ADDRESS
The current housing process is manual. This causes delays, missing data, and lack of control.
It's hard to track housing units or requests clearly.
PROJECT GOALS
To create a digital system to manage housing faster and better. The system should help HR, housing officers,
and other staff work together and handle requests easily.
PROJECT OBJECTIVES AND SCOPE
- Employees can request housing online
- Housing officers can assign units and track assets
- The system shows available units
- Admins can control who sees what
PROPOSED SCHEDULE
6 months (estimated)
PROPOSED BUDGET AND COSTS
To be shared after technical review
POTENTIAL LIMITATIONS AND SETBACKS
- Users need training- Some old data may be hard to move
- Internet and server setup must be ready
